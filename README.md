# It's a simple CLI Todo app in GO 


## commands:

#### list all the todo list

```
 ./todo -list  
```
#### Add the tase to the todo list

```
 ./todo -add [task name] 
```

- ./todo -complete=[index] ---> it's mark the task as a complete 
- ./todo -del=[index]  ---> it's delete the task
