# It's a simple CLI Todo app in GO 


### commands:

#### list all the todo list

```
 ./todo -list  
```
#### Add the task to the todo list

```
 ./todo -add [task name] 
```
#### To mark the task as a complete 

```
 ./todo -complete=[index]
```
#### To delete the task 

```
 ./todo -del=[index]
```
